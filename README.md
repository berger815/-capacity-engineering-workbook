# Capacity Assurance — Supplier Capacity Assessment & Verification

A local-first browser application for supplier capacity assessment, capacity verification, demand and lead-time analysis, staffing, equipment, footprint, recovery scenarios, and executive decision packages.

The GitHub Pages application runs its calculation engine in the browser. Supplier data stays on the device unless the assessor deliberately exports or transfers an assessment file.

Field workflow:

1. Start a new assessment, reopen an assessment file, resume the browser recovery snapshot, or inspect the synthetic Northstar demonstration.
2. Import and reconcile supplier calendars, work areas, people and machines, products, hours-per-part routings, and demand.
3. Calculate capacity, explain governing constraints, test recovery, and download the evidence package.

The legacy v6.86 single-file workbook remains available under `legacy/` as the proven reference implementation. The platform application is the commercial successor.
